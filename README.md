# UBUNTU SERVER UTILS

## Português

Ferramentas desenvolvidas por mim para facilitar o gerenciamento básico de servidores Ubuntu Server 24.04 ou superior, com funções como ajustar data e hora, fuso horário, gerenciar rotas, gerenciar serviços, etc. Feito com menu e seletor.

## English

Tools I created to simplify basic management of Ubuntu Server (24.04 and later), with features like adjusting date and time, time zone, managing routes, managing services, etc. Built with a menu and selector.
